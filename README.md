# 🛒 Cart Service

### Микросервис для управления корзиной пользователя

Сервис обеспечивает гибкое управление покупками, поддерживая как стандартные товары, так и индивидуальные заказы (например, кастомные торты). Реализована логика автоматического истечения срока жизни корзины и глубокая интеграция с инструментами мониторинга.

## 🚀 Основные возможности

### 🛒 Функционал корзины (CRUD)
* **Персонализация:** Получение корзины, привязанной к `user_id`.
* **Гибкое добавление:** Поддержка стандартных позиций (`product_variant_id`) и кастомных изделий (`custom_cake_id`).
* **Управление:** Изменение количества товаров и точечное удаление позиций.
* **Очистка:** Быстрое удаление всей корзины.

### ⚙️ Бизнес-логика и администрирование
* **CartSettings:** Выделенный контроллер для управления глобальными правилами сервиса.
* **Expiration Policy:** Настройка времени жизни корзины через поле `expires_at`.
* **Quantity Limits:** Контроль максимального количества товаров в одной корзине.

---

## 📈 Мониторинг и CI/CD

* **CI/CD:** Реализована поддержка развертывания через **GitHub Actions**.
* **Observability:** Настроен экспорт метрик в **Prometheus** для последующей визуализации в **Grafana**.
